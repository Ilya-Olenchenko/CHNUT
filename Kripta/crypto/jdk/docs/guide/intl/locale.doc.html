Supported Locales

Document Revision Date: 1/27/99

On the JavaTM platform, there does not have to be a single set of supported locales, since each class maintains its own localizations. Nevertheless, there is a consistent set of localizations supported by the Java platform classes. Other implementations of the Java platform may support different locales. Those supported by the Sun's implementation are summarized in the following table. Subsequent releases of the platform may include additional locales.

Locales whose names end with the string "_EURO" are variant locales with euro currency support.
